# Smart Intruder Alert System

An IoT-based Smart Intruder Alert System developed using ESP8266 (NodeMCU), IR Sensor, and Blynk IoT.

## Overview

This project detects an intruder using an IR sensor and instantly sends a notification to a smartphone through the Blynk application.

## Features

- Real-time intruder detection
- Instant mobile notifications
- Wi-Fi enabled using ESP8266
- Low-cost security solution
- Easy to install

## Components Used

- ESP8266 NodeMCU
- IR Sensor Module
- Jumper Wires
- USB Cable
- Blynk IoT Platform

## Software Used

- Arduino IDE
- Blynk IoT
- ESP8266 Board Package

## Working

1. ESP8266 continuously monitors the IR sensor.
2. When an object interrupts the IR beam,
3. ESP8266 detects the intrusion.
4. A notification is sent instantly to the Blynk mobile application.
5. The system continues monitoring after the alert.
